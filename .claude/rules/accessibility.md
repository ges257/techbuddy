# Accessibility Rules (Frontend Only)
- All text: min 18px font size
- All buttons/interactive elements: min 48px touch target
- All colors: min 4.5:1 contrast ratio
- No jargon — plain language only
- Max 3 steps to complete any action
- Voice input and TTS must always be available
- Error messages: friendly, no technical terms, suggest next step
- One action per screen — don't overwhelm
- High-visibility focus indicators on all interactive elements
